## Installation

```sh
# 1. Download the idea-claude-code-gui-0.0.9-beta3.zip file

# 2. In IDEA: Settings → Plugins → Install Plugin from Disk → Select the downloaded idea-claude-code-gui-0.0.9-beta3.zip
```

## Local Development

### 1. Install Frontend Dependencies

```bash
cd webview
npm install
```

### 2. Install claude-bridge Dependencies

```bash
cd claude-bridge
npm install
```

### 3. Run Plugin in Debug Mode

Run in IDEA:
```bash
./gradlew clean runIde
```

### 4. Build Plugin

```sh
./gradlew clean buildPlugin

# The built plugin package will be located in build/distributions/ (~40MB)
```
